Real City Realty - Role Switcher

Choose Your Role

Click one of the buttons below to switch to that role and view the application as that user type:

You can also switch roles using the browser console with these commands:

switchToClient() // Switch to Client role
switchToAgent() // Switch to Agent role
switchToAdmin() // Switch to Admin role
getCurrentRole() // Check current role

Client Features

As a Client, you can:

  • Browse available properties
  • Save favorites
  • Connect with client advisors
  • View your dashboard with recent activity
  • Manage your profile and preferences
